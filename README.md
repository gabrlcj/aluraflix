<h1> ANIMEFLIX </h1>

# <i>aluraflix</i>

Projeto realizado na imersão dev da  Alura.

Uma pequena lista com alguns animes com links clicaveis nas imagens que irão redirecionar o usuario para o site de streaming disponivel para assisti-los.
